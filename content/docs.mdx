# Welcome to Aufa Blog!

Halo! Selamat datang di blog personal saya. Blog ini dibangun menggunakan **Next.js 16**, **Tailwind CSS**, dan **TypeScript**.

## Mengapa saya membuat blog ini?
Saya ingin memiliki tempat untuk mencatat perjalanan saya sebagai *Frontend Developer*, membagikan tutorial, dan mendokumentasikan error/bug yang berhasil saya pecahkan agar bisa bermanfaat bagi orang lain.

### Fitur yang ada di blog ini:
- ⚡ Performa super cepat dengan Next.js App Router
- 🎨 Desain minimalis dan *clean* (terinspirasi dari Notion)
- 🌓 Dark Mode / Light Mode otomatis
- 🏷 Filter artikel berdasarkan Tags

Terima kasih sudah berkunjung, dan semoga artikel-artikel di sini bermanfaat!
